# Software flashen

<Callout emoji="&#x2139;">
	Tijd: 15 minuten
</Callout>

Het is tijd om iets met een beeldscherm te doen. De Pi moet een besturingssysteem hebben, namelijk [Ubuntu Server LTS](https://ubuntu.com/download/raspberry-pi). Om Ubuntu op de SSD te krijgen, gebruiken we het programma [Raspberry Pi Imager](https://www.raspberrypi.com/software/). We gebruiken dezelfde software om eenmalig de Pi in te stellen met Raspberry Pi OS.

## Raspberry Pi OS

We beginnen met het flashen van Raspberry Pi OS naar de microSD. Open de Raspberry Pi Imager en selecteer de volgende twee dingen:

- Raspberry Pi Device: Raspberry Pi 4
- Operating System: Raspberry Pi OS Lite (64-bit). Je vindt het onder "Raspberry Pi OS (other)".
- Storage: microSD.

![Setup voor het flashen van Raspi OS Lite](/rpi-imager-raspberry-pi-os-setup.jpeg)

Wanneer je op "NEXT" klikt, zal het vragen of je de instellingen wilt aanpassen. Klik op "EDIT SETTINGS".

![Configuratiebericht voor het flashen van Raspi OS Lite](/rpi-imager-raspberry-pi-os-config.jpeg)

In de instellingen kun je de gebruikersnaam, het wachtwoord en de SSH-instellingen instellen. Stel de gebruikersnaam in op `pi` en het wachtwoord op `raspberry`.

![Algemene instellingen voor het flashen van Raspi OS Lite](/rpi-imager-raspberry-pi-os-general.jpeg)

Schakel over naar het tabblad "SERVICES" en schakel SSH in.

![Service-instellingen voor het flashen van Raspi OS Lite](/rpi-imager-raspberry-pi-os-services.jpeg)

Klik op "SAVE" en schrijf vervolgens de image naar de microSD-kaart. Zodra dit is voltooid, wordt de microSD-kaart uitgeworpen. Je kunt deze in de Pi steken.

## SSD

Sluit de SSD aan op je computer. Open de Raspberry Pi Imager en selecteer de volgende twee dingen:

- Raspberry Pi Device: Raspberry Pi 4
- Operating System: Ubuntu Server 24.04.1 LTS (64-bit). Je vindt het onder "Other general-purpose OS" en dan "Ubuntu". Installeer geen 24.10, omdat dit geen LTS (Long Term Support) versie is.
- Storage: SSD.

![Setup voor het flashen van Ubuntu Server 24 LTS](/rpi-imager-ubuntu-server-setup.jpeg)

Wanneer je op "NEXT" klikt, zal het vragen of je de instellingen wilt aanpassen. Klik op "EDIT SETTINGS".

Gebruik dezelfde instellingen als voor Raspberry Pi OS, maar stel de gebruikersnaam in op `ubuntu` en het wachtwoord op `ubuntu`.

Klik op "SAVE" en schrijf vervolgens de image naar de SSD. Zodra dit is voltooid, wordt de SSD uitgeworpen.
