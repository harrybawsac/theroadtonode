Aangezien we nu een verse installatie draaien vanaf de SSD, hebben we niet de updates van zojuist. Dus SSH de Pi in en voer het volgende nog een keer uit.

```bash
sudo apt update
```

```bash
sudo apt upgrade -y
```

```bash
sudo reboot now
```
